# Expense Tracker App

A modern full-stack expense tracking application that helps users manage their income and expenses efficiently.  
The app provides real-time budgeting insights, categorized transactions, and a clean dashboard UI for better financial tracking.

[Live Demo](https://expense-tracker-sooty-xi.vercel.app)

## Tech Stack

**Frontend**
- React
- TypeScript
- Tailwind CSS
- Axios
- React Router

**Backend**
- Node.js
- TypeScript
- Express
- MongoDB
- JWT Authentication

## Features

- User authentication (login/register)
- Add, edit, and delete expenses
- Categorize transactions
- Monthly budget overview
- Interactive dashboard with summaries
- Protected routes with JWT
- Responsive UI for all devices
